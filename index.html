Previous work

Inventory System

I created a Network Ownership Dragging system using :SetNetworkOwner. I undertook this project with the aim of emulating the Physics gun from Half-Life 2 and thought it would be a good opportunity to try learn something new. This project required me to understand roblox networking and implement it in a way that lets the client request to claim Network Onwnership over parts. I had to protect the remote from exploiters by ensuring that the client can only Own one item at a time.
